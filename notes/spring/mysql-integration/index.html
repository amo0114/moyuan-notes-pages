MySQL 接入实战教程（中文版） ​

目标读者：已经把 Spring Cloud Alibaba 微服务跑起来（Nacos / Sentinel / OpenFeign 都通了），但还没有接过数据库的学习者。

学习目标：跟完本教程后，你能够独立完成 stock-service 与 order-service 的 MySQL 接入，理解每一行配置和代码"为什么这么写"，并能口头解释乐观锁与悲观锁的差异、事务边界的关键陷阱、Flyway 与 Testcontainers 的价值。

预计耗时：6–10 小时（不含 Docker 镜像下载）。

配套文档：

  • 设计文档：docs/superpowers/specs/2026-05-27-mysql-integration-design.md（讲清楚每个决策为什么这么定）
  • 实施计划：docs/superpowers/plans/2026-05-27-mysql-integration.md（29 个原子任务，本教程的"参考答案"来源）

如何使用本教程：每章都按"目标 → 概念 → 实现 → 关键点 → 踩坑 → 自测"展开。先看完概念再敲代码；敲完代码先自己想"如果改成另一种写法会怎样"，再读"关键点"部分核对你的理解。
